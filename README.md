# 🪆 Матрёшка

> Матрёшка — сибирская душа в каждой вещи. Интернет-магазин сувениров ручной работы (Чита).
> **Сделано в Чите. Сделано с любовью.**

Интернет-магазин на **Node.js + Express + EJS + SQLite**. Серверный рендеринг (хорошо для SEO/ЧПУ), session-аутентификация, bcrypt для паролей, REST API.

## Возможности

- 🛍️ Каталог с фильтрами по категориям и сортировкой
- 🔍 Поиск по каталогу с **автодополнением** (live-подсказки)
- 📄 Карточка товара с галереей, отзывами и микроразметкой Schema.org
- 🛒 Корзина (в сессии) + оформление заказа (доставка, оплата, комментарий)
- 👤 Регистрация / вход (session + bcrypt), личный кабинет с историей заказов
- ⭐ Отзывы с рейтингом (для авторизованных)
- ⚙️ Админка: CRUD товаров (с загрузкой изображений), управление заказами, статистика
- 🔗 REST API (`/api/...`) для товаров, заказов и авторизации
- 📱 Адаптивная вёрстка (mobile-first)
- 🚀 SEO: мета-теги, Open Graph, ЧПУ-ссылки, `sitemap.xml`, `robots.txt`
- 📊 Готовая интеграция Яндекс.Метрики (укажите ID)
- 🖼️ Lazy-loading изображений, SVG-плейсхолдеры на лету

## Запуск

```bash
npm install        # зависимости уже установлены
npm run seed       # создать БД и наполнить демо-данными
npm start          # запуск на http://localhost:3000
# или с авто-перезагрузкой:
npm run dev
```

> Требуется **Node.js 22+** (используется встроенный модуль `node:sqlite`).

## Переменные окружения

| Переменная       | Назначение                              | По умолчанию |
|------------------|-----------------------------------------|--------------|
| `PORT`           | Порт сервера                            | `3000`       |
| `SESSION_SECRET` | Секрет для подписи сессий               | (dev-строка) |
| `YM_ID`          | ID счётчика Яндекс.Метрики               | выключено    |

## Структура

```
server.js          — конфигурация Express
db.js              — схема SQLite + сидинг (node db.js --seed)
models.js          — слой запросов (Products, Reviews, Orders)
middleware/        — auth (сессия, роли)
routes/            — index, auth, catalog, cart, orders, reviews, admin, api, placeholder
views/             — EJS-шаблоны (+ views/admin, + partials)
public/            — css, js, uploads (загруженные изображения)
```

## REST API (кратко)

| Метод | Путь                       | Описание                       |
|-------|----------------------------|--------------------------------|
| GET   | `/api/products`            | Список (фильтры `q`,`category`,`sort`) |
| GET   | `/api/products/:id`        | Товар по id                    |
| GET   | `/api/search/suggest?q=`   | Автодополнение                 |
| POST  | `/api/auth/login`          | Вход (JSON)                    |
| POST  | `/api/auth/logout`         | Выход                          |
| GET   | `/api/auth/me`             | Текущий пользователь           |
| GET   | `/api/orders`              | Заказы пользователя (нужен вход) |
| GET   | `/api/orders/:id`          | Заказ по id                    |
