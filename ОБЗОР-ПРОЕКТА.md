**Папка проекта на вашем компьютере:**

Все файлы ниже лежат внутри неё. Открыть папку: проводник Windows → вставьте в адресную строку `C:\Матррёшка`.

## 1. Что это за сайт

Интернет-магазин сибирских сувениров ручной работы (Чита).
Слоган: «Матрёшка — сибирская душа в каждой вещи». Hero: «Сделано в Чите. Сделано с любовью.»

**Технологии:** Node.js + Express (сервер) · EJS (HTML-шаблоны) · SQLite (база данных) ·
сессии + bcrypt (безопасность). Вёрстка адаптивная (mobile-first).

## 3. Как запустить сайт

В папке `C:\Матррёшка` выполняется (терминал PowerShell):

```powershell
# 1) добавить Node в PATH (у вас он установлен, но не прописан)
$env:Path = "C:\Program Files\nodejs;" + $env:Path

# 2) запустить сервер
node server.js
```

После запуска сайт открывается в браузере: **http://localhost:3000**

> Сайт работает, пока запущен `node server.js`. Закроете окно/выключите ПК — сайт перестанет отвечать.
> Чтобы магазин был доступен из интернета 24/7 и с любого устройства — нужен деплой на хостинг.

## 4. Как войти в админку

1. На сайте справа **вверху, в шапке**, есть значок **⋮** (три точки).
2. Нажмите → откроется окно «Код доступа».
3. Введите код: **`7316`** → вы в админке.

Никаких логинов/паролей для админки нет — только код. Если код не введён, страница `/admin`
для посторонних выглядит как «не найдено» (404), то есть наличие админки не выдаётся.

Адрес админки: **http://localhost:3000/admin**
Сам код задан в файле `routes/admin.js` (переменная `ADMIN_CODE`, по умолчанию `7316`).

## 7. Где хранятся товары и заказы

Всё хранится в файле **`db.sqlite`** на сервере (а не в браузере). Поэтому товар, созданный
в админке, виден всем посетителям, которые открывают этот же сервер. Это обычный файл базы —
его можно скопировать, сделать резервную копию и т.п.

Структура таблиц: пользователи, товары, заказы, позиции заказов, отзывы (как в ТЗ).
